## Data Processing

Results were extracted from `experiments/2026Q1/results/` JSON files. Each experiment is linked to its interactive visualization at `https://simcash-487714.web.app/experiment/{id}`.

### Methodology Notes

- **Cumulative SR** = sum(total_settled) / sum(total_arrivals) across all days
- **Cost delta** = (model_cost - baseline_cost) / baseline_cost × 100%
- **Baselines** run for 1 day (simple) or 25 days (complex) with default policies
- **GLM exclusion**: GLM results for periodic_shocks, large_network, and lehman_month are excluded due to a simulator bug that was present when those experiments ran, compromising the data integrity. However, available GLM runs are shown with this caveat noted.
- **Missing r1 runs**: For liquidity_squeeze and periodic_shocks, r1 experiment files were not generated; only r2 and r3 are available.
